# Stack floating point unit simulator
 
### Overview
![demo](http://i.imgur.com/vVHPt5C.png)

### Floating point number
* Exponent: 7 
* Mantissa: 48

### Commands
* **load** - load to ST0
* **add** - add ST0 to ST1
* **sub** - subtract ST0 from ST1
* **mul** - multiply ST1 and ST0
* **div** - divide ST1 to ST0
* **exch** - swap ST0 and ST1
* **dubl** - dubl ST0
* **pop** - pop ST0
* **exit** - end of work
